# **TIME()**

## **函数说明**

提取时间或日期时间表达式 `expr` 的时间部分并将其作为字符串返回。如果 `expr` 为 `NULL`，则返回 `NULL`。

## **函数语法**

```
> TIME(expr)
```

## **参数释义**

|  参数   | 说明 |
|  ----  | ----  |
| expr  | 必要参数。需要提取时间的 date 或者 datetime 格式的输入值  |

## **示例**

```sql
mysql> SELECT TIME('2003-12-31 01:02:03');
+---------------------------+
| time(2003-12-31 01:02:03) |
+---------------------------+
| 01:02:03                  |
+---------------------------+
1 row in set (0.01 sec)
```
